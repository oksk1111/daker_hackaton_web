# 해커톤 웹 기능 명세

## 공통 (모든 페이지)
- **상단바**: 메인 / hackathons / camp / rankings (이동만 확실히)
- **상태 처리**: 로딩중... / 데이터 없음 / 에러 (3종 보여주기)

## 1) 메인페이지 (/)
- **큰 카드**: 버튼 3개
  - 해커톤 보러가기 → `/hackathons`
  - 팀 찾기 → `/camp`
  - 랭킹 보기 → `/rankings`

## 2) 해커톤 목록 (/hackathons)
- **해커톤 목록**: 리스트로 쭉 표시
  - 제목 / 상태(진행중, 종료) / 태그 / 시작일 / 종료일 / 참가자 수?
- **상세 이동**: 카드 클릭하면 `/hackathons/:slug` 로 이동
- **옵션 필터**:
  - 상태 필터
  - 태그 필터

## 3) 해커톤 상세 (/hackathons/:slug)
- **공통**: 섹션 7개 필수 포함
  - 개요(Overview) / 안내
  - 평가(Eval)
  - 일정(Schedule)
  - 상금(Prize)
  - 팀(Teams)
  - 제출(Submit)
  - 리더보드(Leaderboard)
- **Teams 섹션**:
  - "이 해커톤 팀 구성" 초대 / 수락 / 거절 버튼
  - (옵션) 팀 구성 시, 유의사항 팝업
- **Submit 섹션**:
  - 제출 가이드 보여주기
  - 제출 폼(최소):
    - 메모(notes, optional)
    - 파일(zip) 또는 pdf, csv 등 (해커톤 설정에 따라)
- **Leaderboard 섹션**:
  - 점수 기준: 해커톤 설정에 따라 점수 계산 (투표 or 평가 산식)
  - 참가했으나 제출 내역 없으면 "미제출"로 표기 (순위 제외)

## 4) 팀원 모집 (/camp)
- **특징**: 해커톤과 연결되어 있지 않아도 생성 가능 (해커톤에서도 생성 가능하도록)
- **옵션**: 채팅 또는 '쪽지' 보내기 (있으면 Best!!!)
- **팀 리스트 보기**: 쿼리로 필터링
  - `/camp?hackathon=slug` → 해당 해커톤 팀만 필터링
- **팀 모집글 생성 (폼)**:
  - 팀명 (필수)
  - 소개 (필수)
  - 모집 완료 여부 (isOpen)
  - 모집 포지션 (lookingFor[])
  - 연락 링크 (contactUrl) → 오픈카톡/폼 정도
- **옵션**: 수정 / 모집마감 처리

## 5) 랭킹 (/rankings)
- **목적**: 해커톤 전체 참여/순위 기록에 대한 유저별 랭킹
- **랭킹 테이블**:
  - rank / 닉네임 / points
- **옵션**:
  - 기간 필터 (최근 7일 / 30일 / 전체)
